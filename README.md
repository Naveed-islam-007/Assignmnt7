🧠 KeenKeeper

KeenKeeper is a responsive web application built with React that helps users manage and maintain their personal relationships. It allows users to track when they last contacted their friends, set communication goals, and log interactions like calls, texts, and video chats.

🚀 Features
👥 View and manage a list of friends
📅 Track days since last contact
🎯 Set and monitor relationship goals
⚡ Log interactions (Call, Text, Video)
📜 Timeline to view interaction history
🔍 Filter timeline by interaction type
📊 Analytics page with interaction insights
🔔 Toast notifications for user actions
❌ Custom 404 page for invalid routes
📱 Fully responsive design (mobile, tablet, desktop)



🛠️ Technologies Used
    * React.js
    * React Router DOM
    * Tailwind CSS
    * Recharts
    * React Icons
    * Context API
    * React-spinner
    * React-toastify


    
📂 How It Works
*The app loads friend data from a local JSON file
*Users can click on a friend card to view detailed information
*From the details page, users can log interactions (Call/Text/Video)
*Each interaction is saved and displayed in the Timeline page
*The Stats page visualizes interaction data using a Pie Chart
